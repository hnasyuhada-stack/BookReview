BookReview is a simple and user-friendly web platform that allows users to add, manage, and display book reviews. It was developed using HTML, CSS, JavaScript, PHP, and MySQL, focusing on clean UI design and essential CRUD (Create, Read, Update, Delete) functionality.

---

##  Features

-  Add new book reviews  
-  View all submitted reviews  
-  Edit or update existing reviews  
-  Delete reviews  
-  Clean and responsive UI using HTML & CSS  
-  MySQL database for storing review data  
-  Basic input validation
